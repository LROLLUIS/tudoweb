# Pesquisa de Satisfação - TudoWeb 📊

Este projeto consiste em um script Python desenvolvido para automatizar a coleta de feedbacks de clientes da empresa de marketing **TudoWeb**. O programa utiliza estruturas de repetição e decisão para processar dados de atendimento e gerar um relatório simplificado de satisfação.

## 📋 Sobre o Desafio

O objetivo é realizar uma pesquisa com 50 entrevistados (validada com testes de 10 entradas), coletando nome, idade e uma nota para o atendimento prestado.

### Critérios de Avaliação:
- **1:** EXCELENTE
- **2:** BOM
- **3:** RUIM

Ao final, o sistema exibe o total de respostas "Excelente" e "Ruim".

## 🚀 Tecnologias Utilizadas

* **Python 3** (Lógica principal)
* **Estruturas de Repetição (`for`)**
* **Estruturas de Decisão (`if`, `elif`, `else`)**
